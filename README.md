# Dragon's Hoard 🐉
A fun, beginner-friendly Python command-line game.

## How to Play
1. Run the script: `python dragon_game.py`
2. Decide whether to risk stealing gold.
3. Try to escape before the dragon wakes up!

## Concepts Learned
- Python Functions (`def`)
- While Loops and User Input
- Randomization logic
- Handling global/local variables
